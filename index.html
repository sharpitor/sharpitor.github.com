General features

  • Reactively coupled gui components

    • parameters view/edit
    • 2d drawings view/edit
    • 3d rendering view
      • hardware-accelerated (OpenGl)
      • zoom/pan/rotation
      • photograph juxtaposition
      • light source rotation
      • parts-level colour/texture editing
  • Session history stack
    • undo/redo
    • open/save as XML document
  • Remote server link up
    • auxiliary cloud processing
    • user authentication
    • version monitoring

Additional features specifically geared towards surface-of-revolution structures:

  • editing tool for subdivision into developable strips
  • uv-mapping algorithms for conversion of developable strips to 2d cutting patterns.
  • physical simulation of relaxational response to strain in the structure
