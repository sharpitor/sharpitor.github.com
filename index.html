Minimal C++ toolkit for managing 3d polygonal mesh compositions.

Qt module dependencies: QtGui, QtOpenGl and QtCore

Model

The Polygonal class stores a composite Part list and provides API functionality. The Part class in turn comprises a list of polygonal faces, instances of the Face class.

Rigid body transformations and some geometrical primitives are included in the polygonal::primitives and polygonal::transforms namespaces.

There is support for scene data export in Wavefront Object format (.obj)

View

The GLCanvas class provides OpenGl rendering of the scene with interactive zoom, rotation and pan (mouse or keyboard). Easing response is enabled by default.

Faces declared non-convex are tesselated (without winding rules)

Control

The ColourSettings class provides a dialog to control colouring of individual parts in the scene

Example usage

standalone Qt application

#include <QApplication>
#include "polygonal.h"
#include "polygonal_primitives.h"
#include "polygonal_transforms.h"
int main(int argc, char *argv[])
{
QApplication app(argc, argv);
Polygonal theScene= Polygonal();
Part part1 = Part("slab");
part1 << polygonal::primitives::box(1,0.1,1); // add box with slab-like dimensions to part
polygonal::transforms::translate(&part1, QVector3D(2.0,0,0)); // shift the box from the origin
part1.setColour(QColor(200,200,0)); // colour it orange
Part part2 = Part("tapered cylinder");
part2 << polygonal::primitives::cylinder(0.9,0.5,0.3,30);
theScene << part1 << part2; // put the parts in the scene
QGLWidget *view3d = theScene.create3DView(0);
view3d->show(); // create and open a viewer window
QDialog *colourSettings = theScene.createColoursDialog(view3d);
colourSettings-> show(); // create and open a colour settings dialog
return app.exec();
}
