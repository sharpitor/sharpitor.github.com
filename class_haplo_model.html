Detailed Description

Interface for the parametric MODEL core.

HaploModel can represent any structure in which there features a 'rib manifold' of some sort, e.g. planking on a boat hull, veneers of a lute bowl, the spokes of a hot-air balloon.

A central idea of the Sharpitor Physics framework is to view the rib manifold as a population of haploid individuals undergoing division, culling, homogenization and mutation (variegation). Hence the the'Haplo-' prefix.

Interactive editing of the rib manifold according to this template is looked after by HaploPopMV class.
